# C3: 詳細アジェンダ生成

## 目的
C2のリサーチ結果を基に、10,000～15,000文字の高品質な記事のための詳細な構成案（アジェンダ）を生成します。

## 実例：Xアルゴリズムのアジェンダ

```markdown
## アジェンダ概要
- **記事タイトル**: 【2025年最新】Xアルゴリズム完全攻略ガイド：専門家の詳細調査で解明された「おすすめ」の仕組みとリーチ最大化戦略
- **想定文字数**: 10,000～15,000文字
- **ターゲット読者**: X(旧Twitter)をビジネスやマーケティングに活用する中～上級者、コンテンツクリエイター、企業のSNS担当者
- **SEOキーワード**: Xアルゴリズム 2025, Twitter アルゴリズム, X リーチ, X エンゲージメント, X 動画戦略, X Premium

## 詳細構成

### イントロダクション (約1,000文字)
- **目的**: 読者の関心を引きつけ、記事の重要性と価値を提示する
- **主要ポイント**:
  - 2025年のXが「動画ファースト」と「後悔しないユーザー時間」という新哲学に舵を切ったことの衝撃
  - 2023年のアルゴリズム公開以降、透明性が低下し、多くのクリエイターが混乱している現状
  - 本記事が最新の外部研究や専門家の分析に基づく「羅針盤」となることを宣言
- **含めるべきデータ**: Xからニュースサイトへの参照トラフィックが50%減少

### 第1章: Xアルゴリズム、2025年の大変革 - 新たな「3つの哲学」を理解する (約2,000文字)
- **目的**: 2025年のXアルゴリズムを貫く基本的な考え方を理解させる
- **主要ポイント**:
  1. **最重要方針：「動画ファースト」の徹底**: ネイティブ動画がテキストの10倍配信
  2. **新たな価値基準：「情報的で面白いコンテンツ」の正体**: マスク氏の発言を引用
  3. **究極の目標：「後悔しないユーザー時間」の最大化**: エンゲージメント至上主義からの転換
- **構成意図**: 大きな変化の全体像を提示し、基礎知識を構築

[以下、実際のアジェンダに準拠...]
```

## アジェンダ作成のポイント

### 構成の黄金比
- イントロ: 10%（1,000文字）
- 本論: 80%（8,000～12,000文字）
- 結論: 10%（500～1,000文字）

### 各章の役割
1. **第1章**: 全体像・背景・変化
2. **第2章**: 技術的詳細・仕組み
3. **第3章**: 避けるべき失敗・NG行動
4. **第4章**: 実践的アクションプラン
5. **第5章**: 批判的視点・問題提起

### SEO最適化
- タイトルに主要キーワード
- 各章の見出しにもキーワード配置
- 読者の検索意図に応える構成
